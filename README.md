# VirtualVR

A modern Virtual Reality product landing page built to showcase VR headsets, features, and purchasing options through a clean and responsive UI.

🔗 Live Demo:
https://virtualvr-topaz.vercel.app/

# Overview

VirtualVR is a responsive product landing page designed to present Virtual Reality devices in an engaging and visually appealing format.
The website highlights product specifications, features, and purchase options while focusing on a clean layout and smooth user experience.

This project was created to practice modern frontend development practices, responsive layouts, and component-based UI design.

# Installation
